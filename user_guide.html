Getting Started

First Launch

After installing the application using our one-click installer, you'll be greeted with a splash screen followed by the main dashboard. The first time you launch the application, you'll need to connect your platform accounts.

Connecting Your Accounts

  1. Click on the "Settings" option in the left sidebar
  2. Select the "Platforms" tab
  3. Click "Connect New Platform" to add your StartHawk or CoFoundersLab account
  4. Enter your credentials and select your account type (Free or Paid)
  5. Click "Connect" to establish the connection

Note: The application will automatically detect your account type, but you can manually override it if needed.

Dashboard Overview

The dashboard provides a comprehensive overview of your outreach activities and performance metrics.

Key Components

Quick Stats

At the top of the dashboard, you'll find three key metrics:

  • Total Messages: Number of messages sent across all platforms
  • Response Rate: Percentage of messages that received responses
  • Time Saved: Estimated time saved by using automation

Response Rate Chart

A donut chart showing the proportion of messages that received responses versus those that didn't.

Message Performance

A line chart tracking sent messages and responses over time, helping you identify trends in your outreach effectiveness.

Platform Comparison

A bar chart comparing your performance across different platforms (StartHawk and CoFoundersLab).

Performance Metrics

Progress bars showing your response rate and follow-up effectiveness compared to benchmarks.

Monthly Activity

A line chart showing your overall activity levels throughout the year.

Tip: Hover over any chart element to see detailed information in a tooltip.

Managing Messages

The Messages section allows you to view, track, and manage all your outreach communications.

Message List

The message list displays all your sent and received messages, with the following information:

  • Recipient name and platform
  • Message subject and preview
  • Sent date and time
  • Status (sent, delivered, read, responded)
  • Follow-up status

Message Actions

For each message, you can:

  • View the full conversation thread
  • Send a manual follow-up
  • Schedule a follow-up
  • Mark as important
  • Archive or delete

Filtering and Sorting

Use the filters at the top of the message list to:

  • Filter by platform (StartHawk, CoFoundersLab)
  • Filter by status (sent, responded, unresponsive)
  • Filter by date range
  • Sort by date, platform, or response status

Composing Messages

The Compose section allows you to create new messages and set up automated follow-ups.

Creating a New Message

  1. Click on "Compose" in the left sidebar
  2. Select the target platform (StartHawk or CoFoundersLab)
  3. Choose recipients (individual or batch selection)
  4. Select a template or write a custom message
  5. Use personalization tokens to customize your message (e.g., {first_name}, {company})
  6. Preview your message to see how it will appear to recipients
  7. Schedule sending time or send immediately

Message Templates

Create and save templates for common outreach scenarios:

  • Initial contact
  • Follow-up messages
  • Thank you messages
  • Meeting requests

Follow-up Scheduling

Set up automated follow-ups based on recipient response:

  • Define the number of follow-ups (1-3 recommended)
  • Set the time interval between follow-ups (3-7 days recommended)
  • Create different messages for each follow-up stage
  • Enable automatic cancellation of follow-ups if a response is received

Note for Free Accounts: The system will automatically manage your daily message limits to ensure compliance with platform restrictions.

Analytics & Reporting

The Analytics section provides detailed insights into your outreach performance.

Key Metrics

  • Response Rate: Percentage of messages that received responses
  • Response Time: Average time to receive a response
  • Follow-up Effectiveness: Response rate after follow-ups
  • Platform Comparison: Performance across different platforms
  • Time Saved: Estimated time saved through automation
  • Cost Efficiency: Estimated cost savings based on your hourly rate

Custom Reports

Generate custom reports based on:

  • Date ranges
  • Platforms
  • Message types
  • Response status

Data Export

Export your analytics data in various formats:

  • CSV for spreadsheet analysis
  • PDF for presentations
  • PNG for chart images

Settings & Configuration

The Settings section allows you to customize the application to your needs.

Account Settings

  • Profile Information: Update your name, email, and role
  • Time & Money Settings: Set your hourly rate for cost-saving calculations
  • Security: Change password and enable two-factor authentication

Platform Settings

  • Connected Platforms: Manage your StartHawk and CoFoundersLab connections
  • Account Type Settings: Configure free or paid account handling for each platform
  • Sync Settings: Control how often the application syncs with platforms

Messaging Settings

  • Default Templates: Set up and manage message templates
  • Follow-up Settings: Configure default follow-up behavior
  • Personalization: Manage personalization tokens and defaults

Notification Settings

  • Desktop Notifications: Configure when to receive desktop alerts
  • Email Notifications: Set up email alerts for important events
  • Notification Schedule: Define when notifications are active

Appearance Settings

  • Theme: Choose between dark and light themes
  • Layout: Adjust the dashboard layout to your preference
  • Compact Mode: Enable for smaller screens or to fit more information

Free vs. Paid Account Management

The Co-Founders Outreach tool intelligently handles both free and paid accounts on StartHawk and CoFoundersLab.

Free Account Features

  • Smart Message Queuing: Automatically respects daily message limits
  • Optimal Timing: Sends messages at the best times to maximize impact
  • Limit Tracking: Monitors your usage against platform limits
  • Priority Management: Ensures your most important messages are sent first

Paid Account Features

  • Unlimited Messaging: No daily limits on message sending
  • Batch Processing: Send to multiple recipients simultaneously
  • Advanced Analytics: Access to more detailed performance metrics
  • Priority Support: Faster handling of your messages on the platforms

Switching Between Account Types

If you upgrade or downgrade your platform accounts:

  1. Go to Settings > Platforms
  2. Select the platform you want to update
  3. Change the account type setting
  4. Save your changes

Note: The application will automatically adjust its behavior based on your account type, ensuring optimal performance while respecting platform limitations.

Troubleshooting

Common Issues

Issue Possible Cause Solution
Application doesn't start Missing dependencies or corrupted installation Reinstall the application or run as administrator
Can't connect to platforms Invalid credentials or network issues Verify your login details and check your internet connection
Messages not sending Platform limits reached or connection issues Check your daily limits or reconnect your platform account
Analytics not updating Sync issues or data processing delay Manually trigger a sync in Settings > Platforms
Application running slowly Too many messages or limited system resources Archive old messages or restart the application

Diagnostic Tools

The application includes built-in diagnostic tools:

  1. Go to Settings > Advanced
  2. Click "Run Diagnostics" to check for common issues
  3. View the diagnostic report for recommended actions

Getting Support
