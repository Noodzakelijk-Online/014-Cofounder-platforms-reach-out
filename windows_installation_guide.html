Co-Founders Outreach - Windows 11 Installation Guide

This guide will walk you through the simple one-click installation process for the Co-Founders Automated Reach Out Tool on Windows 11.

System Requirements

  • Windows 10 or Windows 11 (64-bit)
  • 4GB RAM minimum (8GB recommended)
  • 500MB free disk space
  • Internet connection for platform integration

Installation Instructions

Step 1: Download the Installer

Locate the Co-Founders-Outreach-Setup-1.0.0.exe file that you've received.

Step 2: Run the Installer

Double-click the .exe file to start the installation process. If a User Account Control (UAC) prompt appears, click "Yes" to allow the installation.

Step 3: Wait for Installation to Complete

The installer will automatically install the application without requiring any additional input. A progress bar will show the installation status.

Step 4: Launch the Application

Once installation is complete, the Co-Founders Outreach application will launch automatically. You can also find it in your Start menu or on your desktop.

Note: On first launch, Windows may display a security warning. This is normal for new applications. Click "More info" and then "Run anyway" to proceed.

Using the Dashboard

After installation, you'll be greeted with the Co-Founders Outreach dashboard, which provides a comprehensive overview of your outreach activities:

Dashboard Features

  • Analytics Overview: View response rates, message statistics, and platform performance
  • Recent Activity: Track your latest messages and responses
  • Time Saved: See how much time and money the tool has saved you
  • Platform Integration: Connect to StartHawk and CoFoundersLab

Settings Configuration

Click on the "Settings" option in the left sidebar to configure:

  • Account Settings: Update your profile information and security settings
  • Platform Settings: Connect and configure your StartHawk and CoFoundersLab accounts
  • Messaging Settings: Set up templates and follow-up preferences
  • Notification Settings: Configure how you want to be notified of responses

Troubleshooting

Common Issues

  • Application doesn't start:

    Make sure you have the latest Windows updates installed. Try running the application as administrator by right-clicking the shortcut and selecting "Run as administrator".

  • Platform connection issues:

    Verify your internet connection and ensure your platform credentials are correct in the Settings page.

  • Dashboard not showing data:

    The application may need a moment to connect to the backend services. If the issue persists, try restarting the application.

Support

If you encounter any issues not covered in this guide, please contact our support team at support@cofounders-outreach.com or visit our help center at https://help.cofounders-outreach.com.

Privacy Note: The Co-Founders Outreach tool stores all your data locally on your computer. No data is sent to our servers except when interacting with the StartHawk and CoFoundersLab platforms.
